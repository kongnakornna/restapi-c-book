## 16.1 ภาพรวมของโปรเจกต์

โปรเจกต์แรกของเราคือ **เครื่องคิดเลขบวกเลข** ซึ่งเป็นโปรแกรมที่รับตัวเลขสองจำนวนจากผู้ใช้ แล้วแสดงผลลัพธ์ของการบวก โปรเจกต์นี้เรียบง่ายแต่ครอบคลุมแนวคิดพื้นฐานหลายอย่าง:

- การประกาศตัวแปร (`int` หรือ `double`)
- การรับข้อมูลจากผู้ใช้ (`Console.ReadLine()`)
- การแปลงข้อความเป็นตัวเลข (`int.Parse` หรือ `double.TryParse`)
- การคำนวณทางคณิตศาสตร์ (`+`)
- การแสดงผลลัพธ์ (`Console.WriteLine` ร่วมกับ string interpolation)
- การจัดการข้อผิดพลาดเบื้องต้น (ป้องกันการป้อนข้อมูลที่ไม่ใช่ตัวเลข)

> 💡 **เคล็ดลับ:** แม้จะเป็นโปรเจกต์เล็ก แต่หลักการเดียวกันนี้ใช้ได้กับโปรแกรมใหญ่ ๆ ทุกโปรแกรม

### 16.2.2 ฟังก์ชันการทำงาน

1. **รับข้อมูล** – แสดงข้อความแจ้งให้ผู้ใช้ป้อนตัวเลขที่ 1 และตัวเลขที่ 2
2. **แปลงชนิด** – แปลง string ที่รับมาเป็นตัวเลข (ใช้ `double` เพื่อรองรับทศนิยม)
3. **คำนวณ** – นำตัวเลขสองจำนวนมาบวกกัน
4. **แสดงผล** – แสดงผลบวกในรูปแบบที่อ่านง่าย
5. **ป้องกันข้อผิดพลาด** – ถ้าผู้ใช้ป้อนไม่ใช่ตัวเลข ให้แจ้งเตือนและให้ป้อนใหม่ (หรือจบโปรแกรม)

## 16.3 ขั้นตอนการพัฒนา (Step-by-Step)

### ขั้นตอนที่ 1: สร้างโปรเจกต์ใหม่

```bash
dotnet new console -n SimpleCalculator
cd SimpleCalculator
```

### ขั้นตอนที่ 2: เขียนโค้ดเวอร์ชันพื้นฐาน (ไม่มีการป้องกันข้อผิดพลาด)

```csharp
Console.Write("ป้อนตัวเลขที่ 1: ");
double num1 = double.Parse(Console.ReadLine());

Console.Write("ป้อนตัวเลขที่ 2: ");
double num2 = double.Parse(Console.ReadLine());

double sum = num1 + num2;
Console.WriteLine($"{num1} + {num2} = {sum}");
```

### ขั้นตอนที่ 3: เพิ่มการป้องกันด้วย TryParse

```csharp
Console.Write("ป้อนตัวเลขที่ 1: ");
string input1 = Console.ReadLine();

if (!double.TryParse(input1, out double num1))
{
    Console.WriteLine("ตัวเลขที่ 1 ไม่ถูกต้อง");
    return;
}

Console.Write("ป้อนตัวเลขที่ 2: ");
string input2 = Console.ReadLine();

if (!double.TryParse(input2, out double num2))
{
    Console.WriteLine("ตัวเลขที่ 2 ไม่ถูกต้อง");
    return;
}

double sum = num1 + num2;
Console.WriteLine($"{num1} + {num2} = {sum}");
```

## 16.4 โค้ดเต็มของโปรแกรม

**ตัวอย่างที่ 16.1: เครื่องคิดเลขบวกเลข (รุ่นสมบูรณ์)**

```csharp
// บทที่ 16: โปรเจกต์เครื่องคิดเลขบวกเลข
using System;

namespace SimpleCalculator
{
    class Program
    {
        static void Main(string[] args)
        {
            Console.WriteLine("=== เครื่องคิดเลขบวกเลข ===");
            Console.WriteLine("โปรแกรมนี้ใช้สำหรับบวกเลขสองจำนวน\n");

            // รับตัวเลขที่ 1
            Console.Write("ป้อนตัวเลขที่ 1: ");
            string input1 = Console.ReadLine();

            if (!double.TryParse(input1, out double num1))
            {
                Console.WriteLine("ข้อผิดพลาด: ตัวเลขที่ 1 ไม่ถูกต้อง (ต้องเป็นตัวเลขเท่านั้น)");
                Console.WriteLine("กด Enter เพื่อปิดโปรแกรม...");
                Console.ReadLine();
                return;
            }

            // รับตัวเลขที่ 2
            Console.Write("ป้อนตัวเลขที่ 2: ");
            string input2 = Console.ReadLine();

            if (!double.TryParse(input2, out double num2))
            {
                Console.WriteLine("ข้อผิดพลาด: ตัวเลขที่ 2 ไม่ถูกต้อง (ต้องเป็นตัวเลขเท่านั้น)");
                Console.WriteLine("กด Enter เพื่อปิดโปรแกรม...");
                Console.ReadLine();
                return;
            }

            // คำนวณผลบวก
            double sum = num1 + num2;

            // แสดงผลลัพธ์
            Console.WriteLine("\n=== ผลลัพธ์ ===");
            Console.WriteLine($"{num1} + {num2} = {sum}");
            
            // แสดงผลแบบมีรูปแบบ (ทศนิยม 2 ตำแหน่ง)
            Console.WriteLine($"{num1:F2} + {num2:F2} = {sum:F2}");

            Console.WriteLine("\nกด Enter เพื่อปิดโปรแกรม...");
            Console.ReadLine();
        }
    }
}
```

**ผลลัพธ์ตัวอย่าง:**

```
=== เครื่องคิดเลขบวกเลข ===
โปรแกรมนี้ใช้สำหรับบวกเลขสองจำนวน

ป้อนตัวเลขที่ 1: 15.5
ป้อนตัวเลขที่ 2: 27.3

=== ผลลัพธ์ ===
15.5 + 27.3 = 42.8
15.50 + 27.30 = 42.80

กด Enter เพื่อปิดโปรแกรม...
```

## 16.5 การทดสอบและการดีบัก

### 16.5.1 ทดสอบกรณีปกติ

| ตัวเลขที่ 1 | ตัวเลขที่ 2 | ผลลัพธ์ที่คาดหวัง |
|------------|------------|------------------|
| 10 | 20 | 30 |
| 3.5 | 2.5 | 6.0 |
| -5 | 8 | 3 |
| 0 | 0 | 0 |

### 16.5.2 ทดสอบกรณี error

| การกระทำ | ผลลัพธ์ที่คาดหวัง |
|----------|------------------|
| ป้อน "abc" เป็นเลขที่ 1 | แสดงข้อความ "ตัวเลขที่ 1 ไม่ถูกต้อง" แล้วจบโปรแกรม |
| ป้อน "12x" เป็นเลขที่ 2 | แสดงข้อความ "ตัวเลขที่ 2 ไม่ถูกต้อง" แล้วจบโปรแกรม |

### 16.5.3 การใช้ debugger เพื่อทดสอบ

1. ตั้ง breakpoint ที่บรรทัด `double sum = num1 + num2;`
2. กด F5 และป้อนตัวเลข
3. ดูค่า `num1`, `num2` ใน Watch window
4. กด F10 เพื่อดูผลลัพธ์

## 16.6 การเพิ่มฟีเจอร์เพิ่มเติม (Extension Ideas)

หลังจากทำโปรเจกต์พื้นฐานเสร็จ ลองเพิ่มฟีเจอร์ต่อไปนี้:

1. **ให้โปรแกรมทำงานซ้ำ** – ใช้ `while` loop ให้ผู้ใช้บวกเลขได้หลายครั้งจนกว่าจะพิมพ์ "exit"
2. **รองรับการบวกหลายจำนวน** – รับตัวเลขไปเรื่อย ๆ จนกว่าผู้ใช้จะพิมพ์ "=" แล้วแสดงผลรวม
3. **เลือกเครื่องหมาย** – เพิ่มเมนูให้เลือก +, -, ×, ÷ (จะทำในบทที่ 27)
4. **บันทึกประวัติ** – เก็บผลลัพธ์ที่คำนวณไว้ใน List และแสดงประวัติเมื่อผู้ใช้พิมพ์ "history"

## 16.7 ตารางสรุปสิ่งที่ได้เรียนรู้

| แนวคิด | การประยุกต์ใช้ในโปรเจกต์ |
|--------|--------------------------|
| ตัวแปร `double` | เก็บตัวเลขที่มีทศนิยม |
| `Console.ReadLine()` | รับข้อมูลจากผู้ใช้ |
| `double.TryParse()` | แปลง string → double แบบปลอดภัย |
| การบวก (`+`) | คำนวณผลรวม |
| `if` statement | ตรวจสอบความถูกต้องของข้อมูล |
| String interpolation (`$"..."`) | แสดงผลลัพธ์ |
| `return` | จบโปรแกรมเมื่อเกิดข้อผิดพลาด |

## 16.8 แบบฝึกหัดท้ายบท (3 ข้อ)

🧪 **แบบฝึกหัดที่ 16.1:**  
ปรับปรุงโปรแกรมเครื่องคิดเลขบวกเลขให้ทำงานซ้ำ (ใช้ `while` loop) โดยหลังจากแสดงผลลัพธ์แล้ว ให้ถามผู้ใช้ว่า "ต้องการบวกเลขอีกหรือไม่ (y/n):" ถ้าผู้ใช้พิมพ์ 'y' หรือ 'Y' ให้เริ่มใหม่ ถ้าพิมพ์ 'n' หรือ 'N' ให้จบโปรแกรม

🧪 **แบบฝึกหัดที่ 16.2:**  
เพิ่มการคำนวณลบ คูณ หาร ในโปรแกรมเดียวกัน โดยให้ผู้ใช้เลือกเครื่องหมายก่อน (ใช้ `if-else` หรือ `switch`) จากนั้นรับตัวเลขสองตัวแล้วคำนวณตามเครื่องหมายที่เลือก (ข้อนี้จะทำในบทที่ 27 เช่นกัน แต่ลองทำดูก่อน)

🧪 **แบบฝึกหัดที่ 16.3 (ท้าทาย):**  
เขียนโปรแกรมที่รับตัวเลขหลายจำนวน (คั่นด้วยช่องว่าง) จากผู้ใช้ เช่น "10 20 30 40" แล้วคำนวณผลรวมของตัวเลขทั้งหมด (Hint: ใช้ `string.Split()` และ loop)
