## 14.1 ตัวดำเนินการคืออะไร

**ตัวดำเนินการ (Operator)** คือสัญลักษณ์ที่ใช้ในการดำเนินการกับตัวถูกดำเนินการ (operand) เช่น การบวก การลบ การเปรียบเทียบ หรือการกำหนดค่า ตัวดำเนินการเป็นส่วนประกอบพื้นฐานของทุกนิพจน์ (expression) ในภาษา C#

```csharp
int result = 5 + 3;   // + คือตัวดำเนินการ, 5 และ 3 คือตัวถูกดำเนินการ
bool isEqual = (x == y);  // == คือตัวดำเนินการเปรียบเทียบ
```

## 14.2 ประเภทของตัวดำเนินการใน C#

C# มีตัวดำเนินการหลายประเภท แบ่งตามหน้าที่:

| ประเภท | ตัวดำเนินการ | ตัวอย่าง |
|--------|--------------|----------|
| เลขคณิต (Arithmetic) | `+`, `-`, `*`, `/`, `%` | `a + b` |
| กำหนดค่า (Assignment) | `=`, `+=`, `-=`, `*=`, `/=` | `x += 5` |
| เปรียบเทียบ (Relational) | `==`, `!=`, `<`, `>`, `<=`, `>=` | `age >= 18` |
| ตรรกะ (Logical) | `&&`, `||`, `!` | `isValid && isActive` |
| เพิ่ม/ลดค่า | `++`, `--` | `count++` |
| ระดับบิต (Bitwise) | `&`, `|`, `^`, `~`, `<<`, `>>` | `flags | 0x01` |
| การเข้าถึงสมาชิก | `.`, `[]`, `?.`, `?[]` | `person.Name` |
| การแปลงชนิด | `(type)`, `as`, `is` | `(int)x` |
| อื่น ๆ | `new`, `typeof`, `default`, `await` | `new object()` |

ในบทนี้เราจะเน้นตัวดำเนินการพื้นฐานที่ใช้บ่อย ส่วน Bitwise และอื่น ๆ จะกล่าวถึงเมื่อถึงความเหมาะสม

## 14.3 ตัวดำเนินการทางคณิตศาสตร์ (Arithmetic Operators)

ใช้สำหรับคำนวณทางคณิตศาสตร์กับตัวเลข

| ตัวดำเนินการ | ความหมาย | ตัวอย่าง (a=10, b=3) | ผลลัพธ์ |
|--------------|-----------|---------------------|---------|
| `+` | บวก | `a + b` | 13 |
| `-` | ลบ | `a - b` | 7 |
| `*` | คูณ | `a * b` | 30 |
| `/` | หาร | `a / b` | 3 (int) หรือ 3.333 (double) |
| `%` | เศษ (modulo) | `a % b` | 1 |

**ข้อควรระวังเรื่องชนิดข้อมูล:**

```csharp
int intDiv = 10 / 3;        // 3 (ตัดเศษ)
double doubleDiv = 10.0 / 3; // 3.333333...
double mixed = 10 / 3.0;     // 3.333333... (ตัวใดตัวหนึ่งเป็น double)
```

## 14.4 ตัวดำเนินการกำหนดค่า (Assignment Operators)

ใช้สำหรับกำหนดค่าให้ตัวแปร

| ตัวดำเนินการ | ความหมาย | ตัวอย่าง | เทียบเท่า |
|--------------|-----------|----------|-----------|
| `=` | กำหนดค่า | `x = 5` | `x = 5` |
| `+=` | บวกแล้วกำหนด | `x += 3` | `x = x + 3` |
| `-=` | ลบแล้วกำหนด | `x -= 2` | `x = x - 2` |
| `*=` | คูณแล้วกำหนด | `x *= 4` | `x = x * 4` |
| `/=` | หารแล้วกำหนด | `x /= 2` | `x = x / 2` |
| `%=` | หารเอาเศษแล้วกำหนด | `x %= 3` | `x = x % 3` |

**ตัวอย่าง:**

```csharp
int count = 10;
count += 5;   // count = 15
count *= 2;   // count = 30
count /= 3;   // count = 10
```

## 14.5 ตัวดำเนินการเปรียบเทียบ (Relational/Comparison Operators)

ใช้สำหรับเปรียบเทียบค่าสองค่า ได้ผลลัพธ์เป็น `bool` (true/false)

| ตัวดำเนินการ | ความหมาย | ตัวอย่าง (a=5, b=3) | ผลลัพธ์ |
|--------------|-----------|---------------------|---------|
| `==` | เท่ากัน | `a == b` | false |
| `!=` | ไม่เท่ากัน | `a != b` | true |
| `>` | มากกว่า | `a > b` | true |
| `<` | น้อยกว่า | `a < b` | false |
| `>=` | มากกว่าหรือเท่ากัน | `a >= 5` | true |
| `<=` | น้อยกว่าหรือเท่ากัน | `b <= 3` | true |

```csharp
int score = 85;
bool isPass = score >= 50;     // true
bool isPerfect = score == 100; // false
```

⚠️ **ข้อควรระวัง:** อย่าใช้ `=` (assign) แทน `==` (compare) ในเงื่อนไข เพราะคอมไพล์ผ่านแต่ความหมายผิด

```csharp
if (x = 5)  // Error: ไม่สามารถใช้ assignment ในเงื่อนไขโดยตรง (ยกเว้น bool)
if (x == 5) // ถูกต้อง
```

## 14.6 ตัวดำเนินการเชิงตรรกะ (Logical Operators)

ใช้กับ `bool` สำหรับการตัดสินใจที่ซับซ้อน

| ตัวดำเนินการ | ความหมาย | ตัวอย่าง | ผลลัพธ์ |
|--------------|-----------|----------|---------|
| `&&` | และ (AND) | `true && false` | false |
| `\|\|` | หรือ (OR) | `true \|\| false` | true |
| `!` | นิเสธ (NOT) | `!true` | false |

**ตัวอย่าง:**

```csharp
int age = 20;
bool hasLicense = true;
bool canDrive = age >= 18 && hasLicense;  // true
bool cannotDrive = !hasLicense;           // false

bool isWeekend = true;
bool isHoliday = false;
bool canSleepLate = isWeekend || isHoliday; // true
```

> 💡 **Short-circuit evaluation:** `&&` และ `||` จะประเมินเฉพาะเท่าที่จำเป็น ถ้าฝั่งซ้ายของ `&&` เป็น false จะไม่ประเมินฝั่งขวา เพราะผลลัพธ์เป็น false แน่นอน

```csharp
bool result = (x > 10) && (y++ < 5);  // ถ้า x<=10 แล้ว y จะไม่ถูกเพิ่มค่า
```

## 14.7 ตัวดำเนินการเพิ่มค่าและลดค่า (Increment/Decrement)

ใช้กับตัวแปรตัวเลขเพื่อเพิ่มหรือลดทีละ 1

| ตัวดำเนินการ | ความหมาย | รูปแบบ prefix | รูปแบบ postfix |
|--------------|-----------|---------------|----------------|
| `++` | เพิ่ม 1 | `++x` (เพิ่มก่อนใช้ค่า) | `x++` (ใช้ค่าก่อนแล้วค่อยเพิ่ม) |
| `--` | ลด 1 | `--x` | `x--` |

**ตัวอย่าง:**

```csharp
int a = 5;
int b = ++a;  // a = 6, b = 6 (เพิ่มก่อนแล้ว assign)
int c = 5;
int d = c++;  // d = 5, c = 6 (assign ก่อนแล้วค่อยเพิ่ม)

Console.WriteLine($"a={a}, b={b}, c={c}, d={d}"); // a=6, b=6, c=6, d=5
```

## 14.8 ลำดับความสำคัญของตัวดำเนินการ (Operator Precedence)

เมื่อมีตัวดำเนินการหลายตัวในนิพจน์เดียวกัน C# จะประเมินตามลำดับความสำคัญ (Precedence) ก่อนหลัง ดังนี้ (เรียงจากสูงสุดไปต่ำสุด):

| ระดับ | ตัวดำเนินการ | หมวดหมู่ |
|-------|--------------|----------|
| 1 | `x++`, `x--` | Postfix |
| 2 | `++x`, `--x`, `!`, `(type)` | Unary |
| 3 | `*`, `/`, `%` | Multiplicative |
| 4 | `+`, `-` | Additive |
| 5 | `<`, `>`, `<=`, `>=` | Relational |
| 6 | `==`, `!=` | Equality |
| 7 | `&&` | Logical AND |
| 8 | `\|\|` | Logical OR |
| 9 | `=`, `+=`, `-=`, `*=`, `/=` | Assignment |

**ตัวอย่างการทำงาน:**

```csharp
int result = 5 + 3 * 2;   // คูณก่อน: 3*2=6, แล้วบวก 5 → 11
int result2 = (5 + 3) * 2; // วงเล็บเปลี่ยนลำดับ → 16
```

## 14.9 การใช้วงเล็บเพื่อกำหนดลำดับ

วงเล็บ `( )` มีลำดับความสำคัญสูงสุดเสมอ ใช้เมื่อต้องการให้บางส่วนของนิพจน์ถูกประเมินก่อน

```csharp
double average = (a + b + c) / 3;
bool isValid = (age >= 18) && (hasConsent || isEmergency);
```

> ⭐ **หัวข้อสำคัญ:** ถึงแม้จะรู้ลำดับความสำคัญ ก็ควรใช้วงเล็บเพื่อความชัดเจน โดยเฉพาะเมื่อมีตัวดำเนินการหลายประเภทในนิพจน์เดียว

## 14.10 ตัวอย่างโค้ดที่รันได้จริง

**ตัวอย่างที่ 14.1: การใช้ตัวดำเนินการต่าง ๆ**

```csharp
Console.WriteLine("=== ตัวดำเนินการพื้นฐาน ===");
int x = 10, y = 3;

Console.WriteLine($"x + y = {x + y}");
Console.WriteLine($"x - y = {x - y}");
Console.WriteLine($"x * y = {x * y}");
Console.WriteLine($"x / y = {x / y} (int division)");
Console.WriteLine($"x % y = {x % y} (remainder)");

Console.WriteLine("\n=== การกำหนดค่า ===");
int a = 5;
a += 3;
Console.WriteLine($"a += 3 → {a}");
a *= 2;
Console.WriteLine($"a *= 2 → {a}");

Console.WriteLine("\n=== การเปรียบเทียบ ===");
int score = 75;
Console.WriteLine($"score >= 50: {score >= 50}");
Console.WriteLine($"score == 100: {score == 100}");
```

**ตัวอย่างที่ 14.2: ลำดับความสำคัญและการใช้วงเล็บ**

```csharp
int result1 = 10 + 5 * 2;      // 20
int result2 = (10 + 5) * 2;    // 30

bool b1 = true || false && false;   // true (|| ประเมินทีหลัง &&)
bool b2 = (true || false) && false; // false (วงเล็บเปลี่ยน)

Console.WriteLine($"10 + 5 * 2 = {result1}");
Console.WriteLine($"(10 + 5) * 2 = {result2}");
Console.WriteLine($"true || false && false = {b1}");
Console.WriteLine($"(true || false) && false = {b2}");
```

**ตัวอย่างที่ 14.3: Prefix vs Postfix**

```csharp
int counter = 0;
Console.WriteLine($"counter++ = {counter++}"); // 0 แล้วค่อยเพิ่ม
Console.WriteLine($"ตอนนี้ counter = {counter}"); // 1
Console.WriteLine($"++counter = {++counter}");   // 2 (เพิ่มแล้วใช้)
```

## 14.11 ตารางสรุปตัวดำเนินการและลำดับความสำคัญ

### ตารางที่ 14.1: ตัวดำเนินการพื้นฐานที่ใช้บ่อย

| หมวดหมู่ | ตัวดำเนินการ | ตัวอย่าง |
|----------|--------------|----------|
| เลขคณิต | `+ - * / %` | `sum = a + b` |
| กำหนดค่า | `= += -= *= /= %=` | `total += price` |
| เปรียบเทียบ | `== != < > <= >=` | `if (age >= 18)` |
| ตรรกะ | `&& || !` | `if (isReady && hasTime)` |
| เพิ่ม/ลด | `++ --` | `count++` |

### ตารางที่ 14.2: ลำดับความสำคัญ (จากสูงไปต่ำ)

| ลำดับ | ตัวดำเนินการ |
|-------|--------------|
| 1 (สูงสุด) | `()`, `[]`, `x++`, `x--` |
| 2 | `++x`, `--x`, `!`, `(type)` |
| 3 | `*`, `/`, `%` |
| 4 | `+`, `-` |
| 5 | `<`, `>`, `<=`, `>=` |
| 6 | `==`, `!=` |
| 7 | `&&` |
| 8 | `\|\|` |
| 9 (ต่ำสุด) | `=`, `+=`, `-=`, ฯลฯ |

## 14.12 แบบฝึกหัดท้ายบท (4 ข้อ)

🧪 **แบบฝึกหัดที่ 14.1:**  
จงหาค่าของนิพจน์ต่อไปนี้ (ไม่ต้องเขียนโค้ด) และอธิบายลำดับการคำนวณ:  
ก) `15 - 3 * 4 + 2`  
ข) `(15 - 3) * (4 + 2)`  
ค) `10 / 3 + 5 % 2`  
ง) `true && false || true`

🧪 **แบบฝึกหัดที่ 14.2:**  
เขียนโปรแกรมรับตัวเลข 3 จำนวน แล้วคำนวณค่าเฉลี่ย (เป็น double) โดยใช้ตัวดำเนินการ + และ / และวงเล็บ

🧪 **แบบฝึกหัดที่ 14.3:**  
จงแปลงนิพจน์ต่อไปนี้ให้ใช้ตัวดำเนินการกำหนดค่าแบบสั้น (เช่น `x = x + 5` → `x += 5`):  
ก) `total = total + price`  
ข) `count = count - 1`  
ค) `value = value * 2`

🧪 **แบบฝึกหัดที่ 14.4 (ท้าทาย):**  
เขียนโปรแกรมที่รับคะแนนสอบ (0-100) แล้วแสดงเกรดตามเงื่อนไขโดยใช้ตัวดำเนินการตรรกะร่วมกับ if (แนะนำให้ใช้ if ธรรมดาก่อน แต่ลองใช้ `&&` และ `||` ในเงื่อนไข):  
- 80 ขึ้นไป: A  
- 70-79: B  
- 60-69: C  
- ต่ำกว่า 60: F
